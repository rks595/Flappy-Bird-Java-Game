# Flappy Bird Java Game

A Flappy Bird clone built using Java Swing with real-time animation,
gravity-based movement, collision detection, and score tracking.

## Features
- Gravity-based bird movement
- Pipe generation using timers
- Collision detection
- Score tracking
- Game over and restart functionality

## Technologies Used
- Java
- Java Swing
- AWT

## How to Run
1. Open the project in Eclipse or IntelliJ
2. Run the FlappyBird.java file
